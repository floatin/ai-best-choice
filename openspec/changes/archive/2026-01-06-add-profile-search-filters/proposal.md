# Change: 添加个人资料搜索过滤器

## Why
当前个人资料搜索功能缺乏按角色和团队进行筛选的能力，用户无法快速定位特定角色或团队成员的个人资料，影响工作效率。

## What Changes
- 在个人资料搜索界面添加角色过滤器下拉菜单
- 在个人资料搜索界面添加团队过滤器下拉菜单  
- 支持角色和团队过滤器的组合查询
- 保持向后兼容性，现有搜索功能不受影响
- 添加过滤器状态持久化（可选）

## Impact
- Affected specs: profile, search
- Affected code: 个人资料搜索组件、搜索API接口、前端过滤逻辑
- Database: 可能需要添加角色和团队的索引优化查询性能
